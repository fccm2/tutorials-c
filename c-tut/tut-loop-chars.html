Printing with printf(), from a For loop

file exp8.c:

#include <stdio.h>

int main()
{
  char s2[] = { 'a', 'b', 'c', 'd', 'e', 'f' };
  int c;

  for (c = 0; c < 6; c++) {
    printf("c-char: %c\n", s2[c]);
  }

  return 0;
}

we compile test and run:

$ gcc -o exp8.exe exp8.c
$ ./exp8.exe
c-char: a
c-char: b
c-char: c
c-char: d
c-char: e
c-char: f

s2 is a char array
