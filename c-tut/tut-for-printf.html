Printing with printf(), from a For loop

file exp7.c:

#include <stdio.h>

int main()
{
  int d;

  for (d = 0; d < 6; d++) {
    printf("d-int: %d\n", d);
  }

  return 0;
}

we compile test and run:

$ gcc -o exp7.exe exp7.c
$ ./exp7.exe
d-int: 0
d-int: 1
d-int: 2
d-int: 3
d-int: 4
d-int: 5

you can try to test with <= 6 in-stead of < 6, to see the difference,
